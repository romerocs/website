# Modern Solutions for building a Responsive Hero Banner

This is a solution for a responsive hero that I've started using recently. If the hero banner goes the full width of the browser window, you can use the viewport width unit (vw) as a way to maintain the aspect ratio. And the the `clamp()` function provides a concise way to set the min and max heights. Here's a [post](https://chipcullen.com/css-min-max-clamp-functions/) about css math functions that was helpful for me. You could also just use `min-height` and `max-height` if you need to support IE 11.

```html
<header>
  <div>
    <h1>Title</h1>
  </div>
</header>
```

```css
/* test comment */

header > * {
  display: grid;
  place-content: center;
  height: clamp(420px, 41.3vw, 698px);
}
```

You can see I'm using CSS Grid to center the child content of the header. I typically use Flexbox to center content, but I thought, why not live a little, you deserve it! I just love how you can achieve a centered layout with just those 2 lines.
